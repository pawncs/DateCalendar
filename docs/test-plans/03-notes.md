# 测试流程：笔记

## 前置条件
- 在线模式：运行 `start.bat`（Tauri 桌面应用 + HTTP API :9876 + 浏览器 :5173）
- 离线模式：仅 `npx vite`（SQL.js 降级，数据不持久化）

## 白盒测试（Rust 后端）

```bash
cd datecalendar/src-tauri
cargo test --lib test_save_note_create_and_update test_get_and_delete_notes -- --nocapture
```

### 覆盖用例

| # | 用例 | 验证点 |
|---|------|--------|
| 1 | `test_save_note_create_and_update` | upsert 模式 |
| 2 | `test_get_and_delete_notes` | 查询和删除笔记 |

## 前端黑盒测试（Playwright）

> 以下用例可在在线模式（HTTP API）或离线模式（SQL.js 降级）中执行。在线模式操作真实数据库，离线模式数据仅存内存。

### TC-01: 创建笔记

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 选中一个任务 | 显示任务详情 |
| 2 | 切换到"笔记"标签 | 显示笔记编辑器 |
| 3 | 输入标题和内容 | 输入框填充 |
| 4 | 点击保存 | 笔记创建成功 |

### TC-02: 编辑笔记

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 选中已有笔记的任务 | 笔记列表显示 |
| 2 | 点击笔记进入编辑 | 加载已有内容 |
| 3 | 修改内容并保存 | 更新成功（upsert 模式） |

### TC-03: 删除笔记

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 选中已有笔记的任务 | 笔记列表显示 |
| 2 | 点击删除 | 笔记被移除 |

# 测试流程：悬浮窗窗口创建与屏幕停靠

> 对应设计文档：[D-13 悬浮窗窗口创建与停靠](../design/D-13-floating-window.md)

## 前置条件

- 仅桌面模式：运行 `start.bat`（Tauri 桌面应用 + HTTP API :9876 + 浏览器 :5173）
- 悬浮窗为系统级特性，无法在纯浏览器环境中测试
- 测试环境需为 Windows 单显示器（多显示器为后续兼容）

## 白盒测试（Rust 后端）

```bash
cd datecalendar/src-tauri
cargo test --lib test_create_floating_window test_floating_window_config test_docking_position -- --nocapture
```

### 覆盖用例

| # | 用例 | 验证点 |
|---|------|--------|
| 1 | `test_create_floating_window` | 悬浮窗窗口成功创建，label 为 "floating" |
| 2 | `test_floating_window_config` | 窗口配置：无边框、置顶、透明、跳过任务栏、固定尺寸 340×560 |
| 3 | `test_docking_position_hidden` | 初始位置：窗口主体在屏幕外（x > screenWidth），仅露 8px 边缘 |
| 4 | `test_docking_position_shown` | 显示位置：窗口 x = screenWidth - 344px（贴右边缘，留 4px 间距） |
| 5 | `test_get_cursor_position` | `get_cursor_position` IPC 命令返回有效 {x, y} 屏幕坐标 |
| 6 | `test_window_position_switch` | `set_floating_position(true/false)` 命令正确切换显示/隐藏位置 |
| 7 | `test_primary_monitor_detection` | 正确获取主显示器尺寸和缩放因子 |

## 前端黑盒测试（手动 / Playwright）

> 以下用例需在 Tauri 桌面环境中执行。部分可用 Playwright 验证主窗口状态，悬浮窗需目视确认。

### TC-01: 悬浮窗窗口创建

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 启动 `start.bat` | 主窗口正常打开 |
| 2 | 检查窗口列表（Alt+Tab） | 存在两个窗口：主窗口 + 悬浮窗 |
| 3 | 检查 Windows 任务栏 | 任务栏中仅有主窗口图标，无悬浮窗图标 |
| 4 | 检查任务管理器（Ctrl+Shift+Esc） | DateCalendar 进程存在 |

### TC-02: 悬浮窗初始位置与样式

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 启动应用后，看桌面右侧 | 有一条细边（约 8px）贴在屏幕右边缘 |
| 2 | 细边高度约 560px | 垂直居中于屏幕 |
| 3 | 确认细边无标题栏 | 无边框窗口 |
| 4 | 确认悬浮窗背景透明 | 细边区域能看到桌面背景 |

### TC-03: 窗口置顶

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 打开任意全屏应用（如 VS Code） | 悬浮窗细边仍可见 |
| 2 | 打开多个窗口覆盖 | 悬浮窗始终在最上层 |
| 3 | 在悬浮窗上右击（如可行） | 悬浮窗不失去置顶状态 |

### TC-04: 鼠标靠近 → 窗口滑出

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 鼠标移到屏幕最右边缘 | 等待约 200ms → 悬浮窗从右侧滑出 |
| 2 | 悬浮窗完全展开 | 340px 宽度完全可见，距离右边缘约 4px |
| 3 | 鼠标在悬浮窗内移动 | 悬浮窗保持显示状态 |

### TC-05: 鼠标离开 → 窗口隐藏

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 悬浮窗显示中，鼠标向左移出窗口 | 等待约 3 秒 → 悬浮窗滑回隐藏 |
| 2 | 隐藏后 | 仅露出 8px 边缘 |
| 3 | 鼠标完全在窗口左侧 | 隐藏状态保持 |

### TC-06: 防误触

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 鼠标快速从右边缘划过（<200ms） | 悬浮窗不触发显示 |
| 2 | 鼠标停在右边缘刚好 200ms | 悬浮窗触发显示 |
| 3 | 反复快速划过 5 次 | 悬浮窗始终不触发（不会误触发） |

### TC-07: 窗口垂直居中

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 悬浮窗显示状态 | 窗口上下边距相等 |
| 2 | 改变系统分辨率后重启 | 悬浮窗重新计算并垂直居中 |

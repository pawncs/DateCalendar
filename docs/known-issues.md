# 已知遗留问题

> 记录时间：2026-06-19 | 阶段：Phase 3 (D-13~D-17)

---

## 1. 边缘鼠标弹出自定义失效

- **现象**：悬浮窗隐藏（仅露 24px 边缘）后，鼠标移到右边缘不会自动弹出
- **分析**：Tauri v2 透明窗口（`transparent: true`）可能不接收鼠标事件（hit-testing 问题）。已尝试 `background: rgba(0,0,0,0.01)`，仍不稳定
- **方向**：Rust 侧光标轮询 + 右边缘触发区检测

## 2. 任务栏点击不弹出悬浮窗

- **现象**：点击任务栏 DateCalendar 图标，主窗口恢复但不弹出悬浮窗
- **分析**：Tauri v2 `WindowEvent` 缺少 `Shown` 事件，`Focused(true)` 在最小化恢复时可能不触发
- **方向**：额外事件监听或平台特定钩子

## 3. 透明度循环热键无视觉反馈

- **现象**：`Ctrl+Shift+T` 循环切换透明度后，用户无法确认当前档位
- **影响**：盲操，不知道当前是 85%/60%/40%
- **方向**：前端加 toast 提示或档位指示器

## 4. 悬浮窗多 DPI 垂直偏移

- **现象**：部分显示器上悬浮窗垂直位置不居中
- **分析**：`monitor.scale_factor()` 与 `monitor.size()` 的物理/逻辑像素转换可能不准确
- **方向**：多显示器、多 DPI 场景测试
